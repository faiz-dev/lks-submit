1. Pastikan sudah terinstall composer dan nodejs
2. composer install
3. npm install && npm run build
4. copy .env.example .env
5. ubah konfigurasi database pada .env
6. php artisan key:generate
7. php artisan migrate
8. php artisan shield:super-admin dan masukkan user untuk login admin
9. php artisan shield:publish admin
11. php artisan shield:generate --all
12. jalankan dengan php artisan serve
13. Coba login pada localhost:8000/admin/login dengan akun admin yang tadi dibuat
